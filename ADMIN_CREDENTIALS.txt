## ğŸ”§ Troubleshooting

### Problem: OTP not received

**Solutions:**
1. Check spam/junk folder
2. Verify email configuration in `.env.local`:
   ```
   GMAIL_USER=cnp3301@gmail.com
   GMAIL_APP_PASSWORD=xvnbqkhincedpuvy
   ```
3. Restart server: `npm run dev`
4. Try requesting OTP again

### Problem: "User not found"

**Solution:**
```sql
-- Check if user exists
SELECT * FROM users WHERE email = 'admin@kprcas.ac.in';

-- If not found, run CREATE_ADMIN_USER.sql
```

### Problem: "Invalid user type"

**Solution:**
```sql
-- Update user type to admin
UPDATE users 
SET user_type = 'admin', role = 'admin'
WHERE email = 'admin@kprcas.ac.in';
```

### Problem: "Account inactive"

**Solution:**
```sql
-- Activate account
UPDATE users 
SET status = 'active'
WHERE email = 'admin@kprcas.ac.in';
```
